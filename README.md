# Hugo&Cat Code Test

**Weather Widget**

# Requirements

- Node 16.17.0^
# Setup

- Copy `.env.example` to `.env`
- Update env variables
- `yarn` - installs dependencies

## Commands

- `yarn start` - dev mode
- `yarn test` - run tests
- `yarn build` - build the app
- `yarn build` - build the app
- `yarn stylelint` - Runs style lint
- `yarn stylelint:fix` - Fixes any style issues
